What does Hreflang Pro do?

  
Hreflang Pro generates the <link rel="alternate" hreflang="..."> tags that tell Google which language and regional version of a page to show a user, based on their location and browser language. Add a row per language/region version of your page, and it builds the full, correctly formatted tag set — ready to paste into your site's <head>.

  
Each row lets you set a language, an optional region (or x-default as a fallback for unmatched users), and the URL of that specific version. The generator combines language and region into the correct hreflang value automatically — for example, English for the UK becomes en-gb, while plain English with no region stays en.


  
Frequently asked questions


  

    

      Is Hreflang Pro free to use?
      
Yes, completely free with no account or usage limit.

    

  


  

    

      What does x-default mean?
      
It's a fallback hreflang value shown to users whose language or region doesn't match any of your other tagged versions — commonly pointed at a language-selector page or your default/global version.

    

  


  

    

      Do hreflang tags need to be reciprocal?
      
Yes — Google's guidance is that every page in a hreflang set should reference every other page in the set, including itself (self-referencing). Missing or one-directional references are a common source of hreflang errors, so make sure the same full tag block appears on every language version, not just the "main" one.

    

  


  

    

      Is my URL and language data uploaded anywhere?
      
No. Tag generation happens entirely in your browser with plain JavaScript — nothing is sent to a server.

    

  


  

    

      How many language versions can I add?
      
As many rows as you need — click "+ Add Language Version" for each additional language or region your page has.
